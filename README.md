# 日常项目

个人开发项目集合，主要使用 Java + Spring 技术栈，部分项目通过 AI 辅助开发。

## 项目目录

| 目录 | 说明 | 技术栈 |
|------|------|--------|
| `dormitory-system/` | 校园宿舍管理系统 | Spring Boot 2.5 + MyBatis + MySQL |
| `ecommerce-platform/` | 电商平台 | Servlet + JSP + c3p0 + DBUtils |
| `spring-experiments/` | Spring 生态实验合集 | Spring Framework 5.3 |
| `java-basics/` | Java 基础练习 | JDBC、单元测试 |

### 宿舍管理系统
学生管理、报修工单流转、公告管理、文件上传、防暴力破解登录。

### 电商平台
用户系统、商品分类搜索、购物车、推荐系统、订单结算。

### Spring 实验
- `spring-aop/` — AspectJ 切面日志
- `spring-jdbc-transaction/` — JdbcTemplate + 声明式事务
- `spring-bean-lab/` — 三层分层架构
- `spring-mvc-interceptor/` — 自定义拦截器 + 角色权限
- `spring-mvc-user-crud/` — 用户增删改查

### Java 基础
JDBC 数据库连接、PreparedStatement 查询。
